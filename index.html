Engineered for Developer Productivity

Tired of switching between text editors, terminal windows, Chrome tabs, and password managers? Dev Cockpit merges them into a clean local workflow.

Secrets Masking

Automatically detects database passwords, AWS keys, and API tokens. Shields them behind a password mask and enables secure one-click copying.

Command Runner

Run terminal commands directly from your markdown notes. Watch stdout and stderr stream in real-time in a collapsible terminal drawer.

Port Connection Monitor

Displays real-time socket connection lights (green/red) next to socket details in notes. Instantly verify if your server or database port is running.

Cron Schedulers

Use @every or @at annotations to schedule jobs. Runs in background and posts native macOS banner updates on success.

Git branch Banner

Displays branch names, local uncommitted modifications count, and fast Pull/Push buttons when a tab corresponds to a local git repository.

Biometric Tab Lock

Lock sensitive project tabs behind Touch ID or local passwords. Keeps API credentials encrypted and secure from unauthorized access.

Why does macOS block the app when I double-click it?

When you double-click Dev Cockpit for the first time, macOS will show a security warning and block it from opening. This is completely normal — macOS does this for every app not distributed through the App Store or signed with a paid Apple Developer ID ($99/yr). Dev Cockpit contains no tracking, no telemetry, and no network calls. All your data stays 100% on your machine and is never shared.

Steps 2 & 3 — One-time only
macOS will block the app on first launch — here's how to approve it

This happens once. After approval, the app opens instantly every time from the menu bar.

macOS Tahoe — System Settings › Privacy & Security

1
Mount & drag to Applications

Double-click NoteTaker.dmg, then drag the app icon into your Applications folder.

2
Open Privacy & Security

Go to → System Settings → Privacy & Security. Scroll to the Security section at the bottom.

3
Click "Open Anyway"

You'll see "NoteTaker was blocked…" — click Open Anyway, enter your Mac password. Done!

Saved permanently — app opens instantly from now on.

macOS Tahoe — Finder › Applications › Right-click

1
Mount & drag to Applications

2
Right-click the app

In Applications, right-click (or hold Control and click) on NoteTaker.app.

3
Select "Open"

From the menu, click Open. A dialog appears — click the Open button inside it.
